### Mark Washington&apos;s Hotel Engine Coding Demo

#### Instructions

1. Install Docker.
2. Clone this repository.
3. Go into the directory where the repository is cloned.
4. Run docker-compose up.
5. Go to http://localhost:3000 in a web browser.

#### Tools/Frameworks

* ExpressJS
* ReactJS
* Typescript
* Material-UI
* Redux
* Redis
* React-Router
* Thunk
* Axios
* Babel
* Docker
* Nginx
